Программа вычисления Хэш-функции для файлов.
Состоит из исполнимого файла
Hash_Calc_peN02nEDBNjbGKoHUJHxZtEXVeSSztaCaMEyGdF4U=.exe
и исходного файла 
Hash_Calc_pkAJDN0mv0nuQPjAwEIO09tr0SDRHKBSnxqomg4zPI=.cpp



Правила использования:
	Для пользователей Windows.
	1) Скопируйте файл Hash_Calc_peN02nEDBNjbGKoHUJHxZtEXVeSSztaCaMEyGdF4U=.exe 
	в директорию где нахоится файл для которого необходимо вычислить хэш-значение.

	2) Вызовите .exe файл нажав на него.

	3) Выберите номер, соответствующий самому исполнимому файлу и проверьте, что для него 
	результатом вычислений является "peN02nEDBNjbG/KoH/UJHxZtEXVeSSztaCaMEyGdF4U=".

	4) Если на шаге 3 результат отличается от искомого -- обратитесь к преподавателю за помощью.
	
	5) Заново вызовите исполнимый файл и выбирите номер соответствующий файлу для которого нужно вычислить значение функции.






	Для пользователей Linux
	1) Скопируйте файл Hash_Calc_pkAJDN0mv0nuQPjAwEIO09tr0SDRHKBSnxqomg4zPI=.cpp 
	в директорию где нахоится файл для которого необходимо вычислить хэш-значение.

	2) Откройте коммандную строку и перейдите в директорию куда был скопирован .cpp файл.

	3) Выполните команду 
	g++ Hash_Calc_pkAJDN0mv0nuQPjAwEIO09tr0SDRHKBSnxqomg4zPI=.cpp -o Hash_Calc_RDDsj2kHbjHCcYzntr2SyOlXGKu9Pi8haCtJWtjjlM=

	4) Если команда не выполнена установить g++ , для этого введите команды:
	sudo apt update
	sudo apt install g++
		и повторите шаг 3

	5) Для вызова программы используйте комманду 
	./Hash_Calc_RDDsj2kHbjHCcYzntr2SyOlXGKu9Pi8haCtJWtjjlM=

	6) Проверьте, что для .cpp файла результатом является "pkAJDN0mv0nuQPjAwEIO09tr0SDRH/KBSnxqomg4zPI="

	7) Если на шаге 6 результат отличается от искомого -- обратитесь к преподавателю за помощью.
	
	8) Заново вызовите исполнимый файл и выбирите номер соответствующий файлу для которого нужно вычислить значение функции.


	Для пользователей иных платформ предлагается установить том или ином формате коммандную строку bash и посторить схему 
	предложенную для пользователей Linux.
